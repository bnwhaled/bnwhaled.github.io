## <문제점>

1. 개념 숙지 부족
2. 코테에서 변수할당 이해부족

## <시도한 것>

1. 모르는 것 위주로 개념정리 == day1(1)
2. 할당이 되는 부분을 vscode script태그에 넣어 console.log로 찍어보기

## <해결> 

1. 개념에 대해 숙지가 부족하지만 꾸준히 복습할 것
2. 비록 직접 풀어낸 것은 아니지만 어떤 식으로 코드가 돌아가는지 알아 참고하여 직접 작성해봄



## <새롭게 알게된것>

1. prompt 함수

   ```
   prompt('질문','출력 or 사용자 입력값') 으로 모달창 띄우기 가능
   ```

2. 형변환

   String(1) 문자열로

   Number("1") 숫자형으로

   alert(1+3) alert는 문자형으로 값을 받지만 숫자의 연산 적용시 숫자형으로 변경

3. 비교연산자

   비교연산자 사용시 불린형 반환
   문자열 비교시 사전순으로 뒤에 있는 것이 큰 것
